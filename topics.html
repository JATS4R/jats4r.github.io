We use the following conventions:

  • Topic names are lowercase, hyphen-delimited
  • GitHub issue labels that correspond to topics match the topic names exactly, and therefore begin with a lowercase letter. Those that don't (e.g. "Duplicate") begin with an uppercase letter.
